## 📌 오늘 배운 핵심
- `print()` → 출력하기  
- `변수` → 값을 저장하는 상자  
- `input()` → 사용자 입력 받기  
- `int()` → 입력값을 숫자로 변환  
- `if ~ else` → 조건에 따라 다른 코드 실행
